# ExcelSheetSplitter
A free way to split your excel file's sheets into seperated excel files.

#### 对于财务工作者来说，Excel 是高频使用的软件。将工作簿中的工作表拆分成独立的 Excel 文件，更是经常性需求。<br>

#### 然而，Office Excel 中并不自带这样的功能，只能借助 Kutools 这类收费插件达成，WPS 虽有这样的功能，但也需要开通会员。<br>

#### 近期因工作变动，再三思索下，开始了这个项目，分享给同事，亦分享给互联网上的大家。<br>

-----------------

### 使用说明：<br>

#### 软件支持macOS及Windows<br>

下载对应版本后，选中需要拆分的文件（macOS 暂不支持拖拽）；<br>
选中需要拆分的工作表；<br>
选择导出路径；<br>
可选是否要将公示转换成数值。<br>

-----------------
